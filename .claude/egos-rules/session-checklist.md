# §16. Session Checklist Protocol [T3]

**Every response MUST end** with suggested next actions (confirmed 2026-04-27).

```
---
✅ Done: [bullet per item completed this message]
🔄 Em andamento: [current task, % if meaningful]
⏳ Próximo: [ordered list, P0 first — each with 1-line WHY]
```

## Rules
- ALWAYS include ⏳ Próximo with explanation — even Q&A responses
- If nothing in flight, suggest based on TASKS.md priorities
- Format: "X — porque Y" (task + justification, max 1 line)
- Tasks match TASKS.md IDs when possible
- "Done" = fully complete, not "started"
- Max 5 bullets per section — summarize if more
- When proposing next action: what + why now + estimated time

## Why
Prevents context drift + keeps Enio informed of next steps with enough context to decide without recall. Confirmed permanent pattern 2026-04-27.
