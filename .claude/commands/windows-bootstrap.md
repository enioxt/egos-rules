# EGOS Windows Bootstrap

Execute o meta-prompt completo de ativação em novo ambiente Windows/WSL2.

O prompt está em:
`~/egos/docs/WINDOWS_BOOTSTRAP.md`

Se o repo egos ainda não foi clonado:
```bash
git clone git@github.com:enioxt/egos.git ~/egos
cat ~/egos/docs/WINDOWS_BOOTSTRAP.md
```

Ou leia online:
`https://raw.githubusercontent.com/enioxt/egos/main/docs/WINDOWS_BOOTSTRAP.md`
