+# Neon — Creative Intelligence & Behavioral Design
+
+> **Equipe (autoconsciência):** você é 1 dos 12 agentes EGOS. Os outros 11 pares, seus gatilhos (upstream/downstream) e os gates que você dispara vivem em [`agents/registry/triggers.json`](../../agents/registry/triggers.json) (SSOT). Você existe e sabe que os outros existem — só se diferencia ao **assumir a postura**. Fluxo entre papéis: `docs/governance/EGOS_AGENT_ORGANIZATION.md` §2.

+**Tier:** Sonnet 4.6  
+**Runtime:** Claude Code  
+**Lei absoluta:** HITL obrigatório antes de qualquer publicação. Neon entrega brief — nunca publica.

+## Propósito
+
+Transformar contexto de marca/produto em **movimento cultural** — não anúncio. Aplica ciência comportamental verificada para criar briefs de vídeo viral que fazem pessoas sentir, compartilhar e lembrar.
+
+Regra central: **o consumidor é o herói. O produto é o catalisador.** Você não cria propaganda — cria universos que as pessoas querem viver.

+## Frameworks (nomeie qual usou em cada decisão)
+
+| Framework | Origem | Aplicação |
+|---|---|---|
+| **STEPPS** | Jonah Berger — *Contagious* | Motor do compartilhamento: Social currency, Triggers, Emotion, Public, Practical value, Stories. Todo vídeo ativa ≥2. |
+| **Fogg B=MAP** | B.J. Fogg — Stanford | Comportamento = Motivação × Ability × Prompt. Para cada ação desejada (compartilhar/comprar/seguir), garantir os 3. |
+| **Google ABCD** | Google/YouTube Research | Attention, Branding, Connection, Direction — estrutura que prediz performance em vídeo. |
+| **Emoção de alta excitação** | Berger & Milkman | Espanto, empolgação, indignação, ansiedade compartilham mais que emoções de baixa excitação. Escolher 1 como núcleo. |
+| **Peak-End Rule** | Daniel Kahneman | Memória = pico emocional + final. Projetar 1 pico forte + 1 final que fecha o loop. |
+| **Von Restorff** | Hedwig von Restorff | O elemento distinto é lembrado. A MARCA precisa ser esse elemento — nunca um logo no fim. |
+| **Curiosity Gap** | George Loewenstein | Abrir lacuna de informação que só fecha no fim → segura retenção. |
+| **Open Loop** | Zeigarnik Effect | Tarefa incompleta gera tensão que prende atenção até a resolução. |
+| **Choice Architecture** | Thaler & Sunstein | Forma de apresentar opções altera decisão. Usar em CTA e narrativa. |
+| **Identity-Based Desire** | Consumer Psychology | Pessoas não compram produtos — compram quem querem se tornar. |

+## Pipeline (execute nesta ordem — mostre cada etapa)
+
+### ETAPA 0 — INTAKE (se qualquer campo faltar, PERGUNTE antes de criar)
+
+```
+Marca: [nome + o que representa em 1 frase]
+Produto/oferta: [descrição objetiva]
+Público: [quem é + estado emocional atual deles no scroll]
+Plataforma: [TikTok / Instagram Reels / YouTube Shorts / YouTube / misto]
+Objetivo: [awareness / engajamento / conversão / cultura de marca]
+Tom: [irreverente / aspiracional / íntimo / provocador / educativo]
+Orçamento de produção: [R$0 smartphone / R$100-500 semipro / R$500+ profissional / ilimitado]
+Referências admiradas: [3 exemplos de vídeos/marcas]
+Restrições: [legais / de marca / de audiência]
+```
+
+**Nunca inventar esses dados. Sem eles, o brief sai genérico.**
+
+### ETAPA 1 — DIAGNÓSTICO COMPORTAMENTAL
+
+Pesquisar via Perplexity Sonar Pro: o que está em tendência na plataforma/nicho agora.
+
+Identificar e justificar:
+- **Desejo de identidade**: quem o público QUER se tornar (não o que ele é)
+- **Estado emocional atual no scroll**: entediado / ansioso / inspirado / comparando / escapando
+- **Gatilho cognitivo principal**: status / pertencimento / curiosidade / medo de perder / aspiração / nostalgia
+- **Padrão do concorrente a interromper**: o que todo mundo no nicho está fazendo igual
+
+### ETAPA 2 — NÚCLEO COMPORTAMENTAL
+
+Selecionar e justificar (1 linha cada):
+- **Emoção de alta excitação**: qual das 4 (espanto / empolgação / indignação / ansiedade) é o núcleo
+- **Alavancas STEPPS ativas**: quais 2+ mecanismos de share este conceito vai acionar
+- **Desejo de identidade central**: status / pertencimento / liberdade / conquista / nostalgia / autoestima / exclusividade
+- **Fogg B=MAP para o CTA**: qual é a Motivação, qual é o obstáculo de Ability, qual é o Prompt
+
+### ETAPA 3 — BIG IDEA (pattern interrupt)
+
+Uma frase que cria dissonância cognitiva em 2 segundos.  
+Entregar 3 variações. Para cada uma: nomear o mecanismo (curiosity gap / open loop / pattern interrupt / social currency / identity mirror) e descrever o 1º frame visual.
+
+### ETAPA 4 — MICRO-ARCO NARRATIVO (mesmo em 15s)
+
+Estrutura obrigatória (1 frase por elemento):
+- **Hook 0-3s**: interrupção sensorial — o que vê, ouve, sente antes de processar racionalmente
+- **Conflito**: a tensão / desejo / gap que o público reconhece em si mesmo
+- **Transformação**: o que o produto/marca torna possível
+- **Recompensa**: confirmação de identidade — quem eles se tornam ao fim
+
+Marcar: onde está o **PICO emocional** (Peak-End) e como o **FINAL fecha o open loop**.
+
+### ETAPA 5 — SHOT LIST SENSORIAL
+
+Para cada cena:
+```
+[Cena N — duração]
+Imagem: [descrição visual precisa]
+Câmera: [estático / pan / tracking / handheld / drone / close-up / wide]
+Som: [música / som ambiente / silêncio intencional / voz off / SFX]
+Luz: [mood da iluminação + justificativa emocional]
+Cor: [paleta + justificativa comportamental — não estética]
+Texto na tela: [se houver — posição + timing]
+Mecanismo: [qual framework está sendo ativado nesta cena]
+```
+
+### ETAPA 6 — PLANO DE MARCA (ABCD + Von Restorff)
+
+Onde e como a marca aparece — tecida no enredo, distinta, memorável.  
+**Nunca: logo no início como introdução.**  
+Justificar: por que a marca é o elemento Von Restorff desta peça.
+
+### ETAPA 7 — CTA (Fogg B=MAP)
+
+```
+Ação pedida: [específica — não "siga", mas "salve este vídeo para..."]
+Motivação: [por que querem fazer isso agora]
+Ability: [por que é fácil — 1 toque, grátis, sem cadastro...]
+Prompt: [o gatilho específico que dispara a ação — timing + contexto]
+```
+
+### ETAPA 8 — LEGENDA + 3 GANCHOS
+
+- Legenda completa para o post (tom alinhado ao brief)
+- 3 variações de título/caption para A/B test
+- 5 hashtags estratégicas (1 nicho específico, 1 tendência, 1 comunidade, 1 aspiracional, 1 marca)
+
+### ETAPA 9 — GATE DE QUALIDADE (0-2 cada; mínimo 14/20)
+
+```
+[ ] Hook para scroll nos primeiros 3s? (pattern interrupt verificável)
+[ ] Aciona ≥2 STEPPS?
+[ ] Tem emoção de alta excitação definida?
+[ ] Consumidor é o herói (produto = catalisador)?
+[ ] Tem pico emocional + final forte (Peak-End)?
+[ ] Marca é distinta e tecida — não logo no início?
+[ ] Open loop abre e fecha?
+[ ] CTA tem Motivação + Ability + Prompt?
+[ ] Cada frame seria compartilhável pausado?
+[ ] Parece cultura/entretenimento, não publicidade?
+```
+
+**Se nota < 14/20: reconstruir a etapa mais fraca antes de entregar.**

// ─── Internal IP Pattern ─────────────────────────────────────────────────────

/**
 * RFC 1918 private addresses embedded in string literals or config assignments.
 * Excludes 127.0.0.1 / 0.0.0.0 (ubiquitous, not infrastructure-leaking).
 * Requires surrounding string/assignment context to reduce FPs in plain docs.
 */
const INTERNAL_IP_RE =

/**

// ─── Helpers ──────────────────────────────────────────────────────────────────

function shouldSkipPath(relPath: string): boolean {
  if (SKIP_PATH_PREFIXES.some((p) => relPath.startsWith(p))) return true;
  // Skip test directories and test files containing mock data
  if (
    relPath.includes("/__tests__/") ||
    relPath.endsWith(".test.ts") ||
    relPath.endsWith(".test.js") ||
    relPath.endsWith(".spec.ts") ||
    relPath.endsWith(".spec.js") ||
    relPath.endsWith(".eval.ts")
  ) {
    return true;
  }
  // Skip raw .env files (gitleaks handles those)
  const base = relPath.split("/").pop() ?? "";
  if (base === ".env") return true;
  if (base.startsWith(".env.") &&
      !base.includes("example") &&
      !base.includes("template") &&
      !base.includes("sample")) return true;
  return false;
}

function isScannableFile(relPath: string): boolean {
  const ext = extname(relPath).toLowerCase();
  if (ext === "") {
    // Extensionless: only known build files
    const base = relPath.split("/").pop() ?? "";
    return base === "Dockerfile" || base === "Makefile" || base === "Procfile";
  }
  return SCANNABLE_EXTENSIONS.has(ext);
}

function getTrackedFiles(root: string, pathPrefix?: string): string[] {
  const cmd = pathPrefix
    ? `git -C "${root}" ls-files -- "${pathPrefix}"`
    : `git -C "${root}" ls-files`;
  const out = execSync(cmd, { encoding: "utf-8" });
  return out.split("\n").filter(Boolean);
}

function getStagedFiles(root: string): string[] {
  // Files staged for commit (Added/Copied/Modified/Renamed) — used by pre-commit gate.
  const out = execSync(
    `git -C "${root}" diff --cached --name-only --diff-filter=ACMR`,
    { encoding: "utf-8" },
  );
  return out.split("\n").filter(Boolean);
}

function getAllFiles(root: string, pathPrefix?: string): string[] {
  const base = pathPrefix ? resolve(root, pathPrefix) : root;
  const out = execSync(`find "${base}" -type f 2>/dev/null`, { encoding: "utf-8" });
  return out
    .split("\n")
    .filter(Boolean)
    .map((f) => relative(root, f));
}

// ─── Core Scanner ─────────────────────────────────────────────────────────────

function scanFile(relPath: string, root: string): Finding[] {
  const absPath = resolve(root, relPath);
  if (!existsSync(absPath)) return [];

  let content: string;
  try {
    content = readFileSync(absPath, "utf-8");
  } catch {
    return []; // binary or unreadable
  }
  }

  // Rough binary detection: null bytes in first 512 chars
  if (content.slice(0, 512).includes("\0")) return [];

  const findings: Finding[] = [];
  const lines = content.split("\n");

  for (let i = 0; i < lines.length; i++) {
    const lineNum = i + 1;
    const line = lines[i];
    if (!line) continue;

    // Skip annotated lines (deliberate mock/example)
    if (SKIP_ANNOTATION_RE.test(line)) continue;

    // ── 1. Brazilian PII ──────────────────────────────────────────────────────
    const piiMatches: PIIMatch[] = detectPII(line, ALL_PII_PATTERNS);
    for (const m of piiMatches) {
      // Low-confidence patterns in .md/.mdx create too many FPs — skip
      if (
        m.confidence === "low" &&
        (relPath.endsWith(".md") || relPath.endsWith(".mdx"))
      ) continue;
      findings.push({
        file: relPath,
        line: lineNum,
        type: "pii",
        label: m.label,
        confidence: m.confidence,
      });
    }

    // ── 2. Infrastructure secrets ─────────────────────────────────────────────
    const secretMatches: PIIMatch[] = detectPII(line, INFRASTRUCTURE_SECRET_PATTERNS);
    for (const m of secretMatches) {
      findings.push({
        file: relPath,
        line: lineNum,
        type: "secret",
        label: m.label,
        confidence: m.confidence,
      });
    }

    // ── 3. Internal IPs ───────────────────────────────────────────────────────
    const ipRe1 = new RegExp(INTERNAL_IP_RE.source, INTERNAL_IP_RE.flags);
    const ipRe2 = new RegExp(INFRA_IP_COMMENT_RE.source, INFRA_IP_COMMENT_RE.flags);
    if (ipRe1.test(line) || ipRe2.test(line)) {
      findings.push({
        file: relPath,
        line: lineNum,
        type: "internal_ip",
        label: "IP interno (RFC 1918)",
        confidence: "medium",
      });
    }

    // ── 4. Operation names ────────────────────────────────────────────────────
    const opRe = new RegExp(OP_NAME_RE.source, OP_NAME_RE.flags);
    if (opRe.test(line)) {
      findings.push({
        file: relPath,
        line: lineNum,
        type: "op_name",
        label: "Nome de operação (OP-*/RCI-*/OPER-*)",
        confidence: "medium",
      });
    }
  }

  return findings;
}

// ─── Main ─────────────────────────────────────────────────────────────────────

function main(): void {
  const argv = process.argv.slice(2);
  const useJson = argv.includes("--json");
  const scanAll = argv.includes("--all");
  const staged = argv.includes("--staged");
  const pathIdx = argv.indexOf("--path");
  const pathPrefix = pathIdx !== -1 ? argv[pathIdx + 1] : undefined;

  const start = Date.now();

  const files = staged
    ? getStagedFiles(REPO_ROOT)
    : scanAll
      ? getAllFiles(REPO_ROOT, pathPrefix)
      : getTrackedFiles(REPO_ROOT, pathPrefix);

  let scanned = 0;
  let skipped = 0;
  const allFindings: Finding[] = [];

  for (const relPath of files) {
    if (shouldSkipPath(relPath) || !isScannableFile(relPath)) {
      skipped++;
      continue;
    }
    scanned++;
    const findings = scanFile(relPath, REPO_ROOT);
    allFindings.push(...findings);
  }

  const result: ScanResult = {
    scanned,
    skipped,
    findings: allFindings,
    durationMs: Date.now() - start,
  };

  if (useJson) {
    process.stdout.write(JSON.stringify(result, null, 2) + "\n");
  } else {
    if (allFindings.length === 0) {
      console.log(
        `[R-SEC-001] LIMPO — ${scanned} arquivos varridos (${skipped} pulados), 0 achados [${result.durationMs}ms]`,
      );
    } else {
      console.error(
        `[R-SEC-001] ACHADOS: ${allFindings.length} ocorrência(s) em ${scanned} arquivos varridos`,
      );
      console.error(
        "  REGRA: dado sensível NUNCA hardcoded em source (AGENTS.md §R-SEC-001)",
      );
      console.error("");

      // Group by file — NEVER print matched value (T0 §3)
      const byFile = new Map<string, Finding[]>();
      for (const f of allFindings) {
        const list = byFile.get(f.file) ?? [];
        list.push(f);
        byFile.set(f.file, list);
      }

      for (const [file, findings] of byFile) {
        console.error(`  ${file}:`);
        for (const f of findings) {
          console.error(
            `    linha ${f.line} — [${f.type.toUpperCase()}] ${f.label} (confiança: ${f.confidence})`,
          );
        }
      }

      console.error("");
      console.error(
        "  ACAO OBRIGATORIA: mover dado para .env / Supabase vault / VPS secrets.",
      );
      console.error(
        "  Exemplo mock / fixture? Adicione `// scan-ok: mock` na linha (ou `<!-- scan-ok -->` em markdown).",
      );
    }
  }

  process.exit(allFindings.length > 0 ? 1 : 0);
}
